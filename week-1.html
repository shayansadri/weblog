ایجاد یک بانک اطلاعاتی و انجام فعالیت های Attach, Detach, Backup, Restore, Import, Export

Attach و detach

با Attach کردن ، فایل دیتابیس قفل میشود و نمیتوان آنرا کپی یا حذف کرد برای آزاد کردن آن اصطلاحا باید Detach انجام دهیم (برعکسAttach) برای اینکار کافیست روی فایل دیتابیس که در زیر شاخه Databases قرار گرفته راست کلیک کرد و از منوی Tasks گزینه Detach را کلیک کنیم و پنجره بعدی را OK نماییم. سپس برای Attach کردن بانک اطلاعاتی که ایجاد کردید روی عبارت Database و سپس Attach کلیک کنید. روی دکمه Add کلیک کنید در پنجره باز شده فایل بانک اطلاعاتی که Detach کردید را انتخاب کنید و روی OK کلیک کنید.

Import & Export

مراحل ایمپورت:روی دیتابیس مورد نظرتان راست کلیک کنید و از Tasks ، گزینه Import Flat File را انتخاب کنید)این گزینه برای فایل های flat مانند فایل های Text یا CSV است (. بدین ترتیب ویزارد مربوطه باز می شود روی Next کلیک کنید. در ادامه در مرحله Specify input file قرار می گیرید که باید دکمه Brows را کلیک کنید و فایل مربوطه را از سیستم تان انتخاب کنید. سپس در قسمت New Table Name ، باید نام یک جدولی که ایجاد می شود و رکوردها در آن ثبت می شود را تایپ و Next کنید. اکنون در بخش preview data قرار دارید در واقع در این بخش شما یک پیش نمایش از دیتایی که قصد import کردن آن به یک Sql server را دارید بنابراین می توانید تأیید کنید که فایل صحیحی را انتخاب کرده اید یا داده های شما در قالب tabluar چگونه به نظر می رسند. سپس روی NEXT کلیک کنید. اکنون به مرحله Modify columns رسیدید اینجا می توانید data type های ستون ها را تغییر دهید و ستون های primary key را مشخص کنید. پس از اعمال تغییرات مورد نظرتان روی Next کلیک کنید. اکنون یک summary یا خلاصه وضعیتی از ورودی های شما مانند نام فایل ورودی ، نام پایگاه داده و نام جدول را فقط برای تأیید اینکه فایل صحیحی را انتخاب کرده اید یا نه، نمایش می دهد.اگر همه چیز درست و صحیح است، FINISH را کلیک کنید. پس از کلیک دکمه finish فرآیند import کردن اطلاعات از فایل به جدول در Sql شروع می شود و بر حسب حجم فایل این فرآیند زمانبر خواهد بود. پس از اتمام Import تیک سبز رنگ همراه با پیغام Operation Complete را دریافت می کنید. مراحل Export ساده است شما فقط باید روی بانک اطلاعاتیی که می خواهید از ان خروجی بگیرید راست کلیک کنید و گزینه Task و سپس Export Data tier – Application را انتخاب کنید. پنجره ای برای شما باز می شود روی Next کلیک کنید و مکانی که میخواهید خروجی بانک اطلاعاتیتان را ذخیره کنید انتخاب کنید. و در اخر یک summary یا خلاصه وضعیتی از خروجی شما نمایش می دهد.اگر همه چیز درست و صحیح استFINISH را کلیک کنید. و منتظر بمانید تا مراحل خروجی تمام شود

Back up & Restore

با ورود به بخش databases بر روی دیتابیس موردنظر کلیک راست کرده و از منوی Task گزینه Backup را انتخاب کنید. با توجه به موارد مشخص شده در تصویر زیر نوع بکاپ را مشخص کنید سپس با کلیک روی پزینه Add مکان ان را مشخص کنید و در اخر روی OK کلیک کنید تا از بانک اطلاعاتیتان پشتیبانی بگیرد. برای بازگردانی یا Restore کردن بانک اطلاعاتی با ورود به بخش databases بر روی دیتابیس موردنظر کلیک راست کرده و از منوی Task گزینه Restore و سپس Database را انتخاب کنید. از قسمت Source گزینه Device را انتخاب کرده و فایل Backupی که پیش‌تر تهیه شده است را با کلیک روی پزینه Add مشخص و انتخاب کنید. و روی گزینه OK کلیک کنید.
